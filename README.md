# vsf-yotpo

## Configuration
### For singlelang
```json
{
  "yotpo": {
    "account_id": "123456",
    "app_key": "aasdasdasdasdas"
  }
}
```
### For multilang
```json
{
  "yotpo": {
    "langs": {
      "en": {
        "account_id": "123456",
        "app_key": "aasdasdasdasdas"
      },
      "es": {
        "account_id": "123456",
        "app_key": "aasdasdasdasdas"
      },
      "de": {
        "account_id": "123456",
        "app_key": "aasdasdasdasdas"
      }
    }
  }
}
```
